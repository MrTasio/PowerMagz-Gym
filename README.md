# PowerMagz - Gym Membership Portal

A modern, mobile-first gym membership portal web application with a clean and intuitive interface.

## Features

- **Login Screen** - Secure authentication with splash screen animation
- **Home Feed** - Social media-style updates from the gym
- **Membership Details** - View membership status and expiration
- **Trainer Directory** - Browse trainers and view detailed profiles
- **Gym Information** - Hours, contact info, rules, and FAQs

## Getting Started

1. Open `index.html` in a web browser
2. Use any email and password to login (demo mode)
3. Navigate through the app using the bottom navigation
